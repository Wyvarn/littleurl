A Little URL Shortener

Oh hi,

Tell me about your long problematic URLs and I will shorten them!

It is simple, pass a URL as a parameter and I will shorten it and give you a JSON response.

Example creation usage

https://glitter-rail.glitch.me/little/https://www.google.com
https://glitter-rail.glitch.me/little/http://foo.com:80

Example creation output

{ "original_url":"http://foo.com:80", "short_url":"https://little-url.herokuapp.com/8170" }

Simple usage

https://little-url.herokuapp.com/2871

Will redirect to:

https://www.google.com/
